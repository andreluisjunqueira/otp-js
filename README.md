> Module to generate a one time password made with pure js, and can be used in any js project !!

## Install

```sh
yarn install onetimepassword-js
```

## Usage

```sh
import otp from 'onetimepassword-js'

const KEY = "your private key"

otp(KEY);
```

## Run tests

```sh
yarn run test
```
